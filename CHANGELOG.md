# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2026-03-13

### Added
- Initial release of JUMPING TUI directory navigation tool
- Three-panel layout for parent-directory/current-directory/preview navigation
- Vim-like keybindings (h/j/k/l) and arrow key support
- Shell wrapper integration for automatic `cd` after exiting
- Cross-platform support (Linux, macOS, Windows)
- CI/CD pipeline with automated testing, building, and releasing
